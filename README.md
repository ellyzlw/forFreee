1. log in: name や　passworkを入力、log inできます。 (完成しました)
2. check in (完成しました) 　顔確認(完成しません)
3. 仕事中:　(完成しません)
   - friends: 他の会社員のリスト。今日、誰が出勤することが確認できます。
